# Comandos CLI

## Ajuda

```bash
sdd master help
```

Mostra os comandos estáveis disponíveis.

## Exemplos

```bash
sdd master init --yes --preset="ecommerce" --ai="codex"
sdd master doctor
sdd master git --json
```

## Update

```bash
sdd master update
sdd master update --dry-run
sdd master update --apply --yes
sdd master update --dry-run --json
```

`update` é funcional, estável e suporta backup, relatório e detecção de conflitos.
