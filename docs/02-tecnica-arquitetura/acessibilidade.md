# Acessibilidade

Acessibilidade é gate obrigatório para projetos com interface.

## Critérios mínimos

- Contraste.
- Navegação por teclado.
- Foco visível.
- Labels.
- Semântica.
- Texto alternativo.
- Feedback não apenas por cor.

## Gate

```bash
sdd master uiux --yes --type="accessibility" --phase="PHASE-01" --profile="WEB"
```
