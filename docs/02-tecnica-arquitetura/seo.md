# SEO

SEO é gate obrigatório para perfis `WEB`, `SAAS` e `E-COMMERCE` quando houver presença web pública.

## Critérios mínimos

- Title.
- Description.
- Headings.
- URLs.
- Conteúdo indexável.
- Schema ou structured data quando aplicável.
- Performance.
- Open Graph.

## Gate

```bash
sdd master uiux --yes --type="seo" --phase="PHASE-01" --profile="WEB"
```
