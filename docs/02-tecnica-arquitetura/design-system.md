# Design System

O SDD Master exige design system antes de frontend relevante.

## Critérios mínimos

- Cores documentadas.
- Tipografia documentada.
- Espaçamento e radius consistentes.
- Componentes base definidos.
- Estados de loading, vazio, erro e sucesso previstos.
- Acessibilidade considerada desde o início.

## Gate

O gate é criado com:

```bash
sdd master uiux --yes --type="design-system" --phase="PHASE-01" --profile="WEB"
```
