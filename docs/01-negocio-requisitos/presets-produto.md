# Presets de Produto — SDD Master

## Objetivo

Presets ajudam o projeto a começar com riscos, documentação e gates coerentes ao tipo de produto.

## Matriz

| Preset | UI/UX | SEO | Foco |
|---|---|---|---|
| web | obrigatório | obrigatório | Aplicação web |
| api | opcional | não aplicável | Contratos e segurança |
| cli | opcional | não aplicável | UX textual e automação |
| mobile | obrigatório | não aplicável | Fluxos nativos |
| desktop | obrigatório | não aplicável | Produtividade e distribuição |
| library | opcional | não aplicável | API pública e semver |
| ecommerce | obrigatório | obrigatório | Catálogo, checkout e conversão |
| generic | opcional | opcional | Projeto genérico |

## Preset ecommerce

O preset `ecommerce` deve priorizar catálogo, checkout, frete, pagamento, segurança, LGPD, acessibilidade, performance, admin, deploy guard e rollback.
