# Código de Conduta

O SDD Master deve ser um projeto profissional, respeitoso e seguro para colaboração.

## Comportamentos Esperados

- Tratar outras pessoas com respeito.
- Discutir ideias com foco técnico e boa-fé.
- Dar feedback claro, específico e construtivo.
- Preservar privacidade e segurança.
- Respeitar o escopo definido para cada fase do projeto.

## Comportamentos Não Aceitos

- Ataques pessoais, assédio ou discriminação.
- Divulgação de dados sensíveis ou privados.
- Pressão para ignorar testes, documentação ou segurança.
- Mudanças fora do escopo combinado sem alinhamento.

## Aplicação

Contribuições que violem este código podem ser recusadas. Em casos graves, o mantenedor poderá restringir participação no projeto.
