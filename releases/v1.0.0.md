# v1.0.0

## Release estável

SDD Master `1.0.0` é a primeira versão estável do projeto.

## Destaques

- `latest` passa a apontar para `1.0.0`.
- `sdd master update` entra no contrato estável.
- README, changelog, checklists e auditorias foram consolidados.

## Publicação

- npm: `npm publish --access public --tag latest`
- instalação recomendada: `npm install -g sdd-master`
- GitHub Release: `v1.0.0`

## Observação

Histórico de prototype, alpha, beta e RC permanece disponível para referência.
