# Release v0.5.0-beta

## Status

Preparada para publicação beta.

## Estágio

Beta prerelease.

## Versão

0.5.0-beta

## Inclui

- Banner próprio do SDD Master na CLI.
- Presets oficiais de projeto.
- Preset `ecommerce`.
- API pública da CLI documentada.
- Contrato público dos comandos.
- Fluxo E2E beta/happy path.
- Documentação beta de presets e comandos.

## Segurança

- `.env` real: não permitido.
- `.sdd-master/` na raiz do pacote: não permitido.
- npm publish deve usar `--tag beta`.
- `latest` não deve ser alterado manualmente.
- GitHub Release deve ser prerelease.

## Instalação após publicação

```bash
npm install -g sdd-master@beta
```
