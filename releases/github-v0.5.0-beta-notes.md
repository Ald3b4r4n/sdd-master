# GitHub Release Notes — v0.5.0-beta

`v0.5.0-beta` consolida o SDD Master como beta operacional para uso real controlado.

## Destaques

- Banner próprio do SDD Master na CLI.
- Presets oficiais: web, api, cli, mobile, desktop, library, ecommerce e generic.
- Preset `ecommerce` com SEO, UI/UX, catálogo, checkout, frete, pagamento, LGPD, acessibilidade, performance, admin, deploy guard e rollback.
- API pública da CLI documentada.
- Contrato público dos comandos.
- Fluxo E2E beta/happy path.

## Instalação

```bash
npm install -g sdd-master@beta
```

Esta é uma prerelease beta, não uma release estável.
