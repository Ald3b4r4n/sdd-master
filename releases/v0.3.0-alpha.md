Alpha

## Inclui

- Tudo de `0.2.0-prototype`.
- Release/deploy guards.
- Implement assistido controlado.
- Registry seguro de plugins/extensões.
- Segurança avançada opt-in.

- `.env` real: não permitido.
- `.sdd-master/` na raiz: não permitido.
- Segredos: não permitidos.
- npm publish real: executado com `npm publish --access public --tag alpha`.
- Git tag: `v0.3.0-alpha` criada e enviada ao remoto.
- GitHub Release: prerelease criada em `https://github.com/Ald3b4r4n/sdd-master/releases/tag/v0.3.0-alpha`.
- dist-tag `alpha`: `0.3.0-alpha`.
- dist-tag `prototype`: `0.2.0-prototype`.
- dist-tag `latest`: `0.1.0-prototype.1`.
